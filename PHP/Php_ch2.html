2.1. Using includes

You can use the "include()" function if you want to reuse parts of a webpage. For instance if you have a header and footer that are the same for every page in your site there is no reason to write the same html code every time you create a new page. It's more easier to write the header and footer sections in two separated file and then use the "include()" function and all the pages will include now the same header and footer.

The best part of this practice is that your is that you modify the header and footer pages all the other pages that have included this files will be automatically updated with the new versions of header and footer.

2.4.1 Rounding numbers with round()

If you want to round a floating number by using the general rules of rounding (i.e. if it is >= .5 round to the next integer number and if it is < .5 round to the previous integer number).

For example 5.14 rounded is 5 while 5.55 rounded is 6.

  • This is how you use the round() function. Keep in mind that the second parameter of the function is how many decimals we want the floating point number to be rounded (for instance 3.14 rounded with 1 decimal will be 3.1) : 
<?php	
    $num_float = 3.14;
	echo round($num_float, 1);	//3.1
	echo "</br>";
	echo round($num_float);	//3
?>

2.5.10. Substring making by taking a starting and ending position with substr()

You can make a substring by taking a starting and ending position with substr():

string substr ( $string , $start [, $length ] )
$string The input string. Must be one character or longer.
$start

If start is non-negative, the returned string will start at the start'th position in string, counting from zero. For instance, in the string 'abcdef', the character at position 0 is 'a', the character at position 2 is 'c', and so forth.

If start is negative, the returned string will start at the start'th character from the end of string.

If string is less than or equal to start characters long, FALSE will be returned.
$length

If length is given and is positive, the string returned will contain at most length characters beginning from start (depending on the length of string).

If length is given and is negative, then that many characters will be omitted from the end of string (after the start position has been calculated when a start is negative). If start denotes the position of this truncation or beyond, false will be returned.

If length is given and is 0, FALSE or NULL an empty string will be returned.

If length is omitted, the substring starting from start until the end of the string will be returned.

$subject The string or array being searched and replaced on, otherwise known as the haystack.
  • Here is how you use the substr() function : 
<?php
	$first = "The quick brown fox";

	echo substr($first, 4, 11);
?>

2.6. Arrays

2.6.1. Defining and accessing arrays

Think of an array like an expandable file folder where you can put items in each of the pockets

There are no limits to how many pockets it can have (not that you'll need to worry about at least)

Then you can reference those items by the pocket number (or "index"). (This is the "value" of that "position" in the array). Be careful! Pockets are numbered starting at 0 (0,1,2,3...) so the index of the 2nd pocket is 1.

  • This is how you define and access arrays : 
<?php 
	// defining a simple array
	$array1 = array(4,8,15,16,23,42);
	
	// referencing an array value by its index
	echo $array1[0];
?>	
<br />
<pre>
<?php
	print_r($array1);
?>
</pre>
<?php	
	// arrays can contain a mix of strings, numbers, even other arrays
	$array2 = array(6,"fox", "dog", array("x", "y", "z"));
	
	// referencing an array value that is inside another array
	echo $array2[3][1];
?>
<br />
<?php

	// Changing values in an array that has already been defined
	// It's just like variables but you use the index to reference the array position
	$array2[3] = "cat";
	echo "<br />";
	echo $array2[3];

?>
<br />
<pre>
<?php
	print_r($array2);
?>
</pre>

2.6.2. Defining and accessing asociative arrays

Asociative arrays are like regular arrays but the values are indexed not by numbers but by strings called keys. Every key identifies uniquely a value and all the values along with their respective keys

  • This is how you define and access asociative arrays : 
<?php
		// You can also assign labels to each pocket (called "keys"),
		$array3 = array("first_name" => "Kevin", "last_name" => "Skoglund");
	
		// which will allow you to use the key to reference the value in that array position.
		echo $array3["first_name"] . " " . $array3["last_name"] . "<br />";
		$array3["first_name"] = "Larry";
		echo $array3["first_name"] . " " . $array3["last_name"] . "<br />";
?>

<br />

A good way to see the values inside an array during development:<br />
<pre><?php print_r($array3); ?></pre>
