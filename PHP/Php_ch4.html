4. Debugging

4.1. var_export() function used to find out class composition

You can debug and display the composition of one object at one time by using the var_export() method

  • Here is how you use var_export() : 
<?php
	class Person {
		public $name;
		public $last_name;
		public $age;	
	}
	
	$pers = new Person();
	
	echo "<h2>Object of type Person with undefined properties</h2>";
	echo "<pre>".var_export($pers, 1)."</pre>";
	
	$pers->name = "John";
	$pers->last_name = "Doe";
	$pers->age = 29;
	
	echo "<h2>Object of type Person with defined properties</h2>";
	echo "<pre>".var_export($pers, 1)."</pre>";
?>
